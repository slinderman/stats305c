Linear Dynamical Systems

Prerequisites

This chapter builds on Hidden Markov Models and Multivariate Normal. The Kalman filter is the continuous-state analogue of the HMM forward algorithm.

The previous chapter introduced Hidden Markov Models, where the hidden state ztz_tzt​ was discrete. Here we keep the same graphical model but allow zt∈RDz\mathbf{z}_t \in \mathbb{R}^{D_z}zt​∈RDz​ to be continuous. When both the dynamics and the emissions are Gaussian and linear, the model is called a Linear Dynamical System (LDS), also known as a linear Gaussian state space model. The key payoff is that the forward-backward algorithm reduces to a pair of closed-form matrix recursions: the Kalman filter Kalman, 1960 and the Rauch-Tung-Striebel (RTS) smoother Rauch et al., 1965.

From HMMs to State Space Models¶

An HMM uses the same graphical model as a general state space model:

p(z1:T,x1:T)=p(z1)∏t=2Tp(zt∣zt−1)∏t=1Tp(xt∣zt).p(\mathbf{z}_{1:T}, \mathbf{x}_{1:T}) = p(\mathbf{z}_1)\prod_{t=2}^T p(\mathbf{z}_t \mid \mathbf{z}_{t-1}) \prod_{t=1}^T p(\mathbf{x}_t \mid \mathbf{z}_t).p(z1:T​,x1:T​)=p(z1​)t=2∏T​p(zt​∣zt−1​)t=1∏T​p(xt​∣zt​).
(1)

In an HMM, zt∈{1,…,K}z_t \in \{1,\ldots,K\}zt​∈{1,…,K} is discrete and the forward-backward algorithm sweeps over KKK values at each time step. Nothing in the graphical model requires discrete states, however. If zt\mathbf{z}_tzt​ is continuous, the sums in the forward-backward recursions become integrals. In general those integrals are intractable, but for the Gaussian linear case they have a closed form.

The Linear Dynamical System¶

The LDS is defined by Gaussian dynamics and Gaussian emissions:

z1∼N(μ0,Σ0)\mathbf{z}_1 \sim \mathcal{N}(\boldsymbol{\mu}_0, \boldsymbol{\Sigma}_0)z1​∼N(μ0​,Σ0​)
(2)
zt∣zt−1∼N(Azt−1+b,  Q)t=2,…,T\mathbf{z}_t \mid \mathbf{z}_{t-1} \sim \mathcal{N}(\mathbf{A}\mathbf{z}_{t-1} + \mathbf{b},\; \mathbf{Q}) \qquad t = 2,\ldots,Tzt​∣zt−1​∼N(Azt−1​+b,Q)t=2,…,T
(3)
xt∣zt∼N(Czt+d,  R)t=1,…,T\mathbf{x}_t \mid \mathbf{z}_t \sim \mathcal{N}(\mathbf{C}\mathbf{z}_t + \mathbf{d},\; \mathbf{R}) \qquad t = 1,\ldots,Txt​∣zt​∼N(Czt​+d,R)t=1,…,T
(4)
SymbolDimensionRole
A\mathbf{A}ADz×DzD_z \times D_zDz​×Dz​Dynamics (transition) matrix
b\mathbf{b}bDzD_zDz​Dynamics bias
Q\mathbf{Q}QDz×DzD_z \times D_zDz​×Dz​Dynamics noise covariance
C\mathbf{C}CDx×DzD_x \times D_zDx​×Dz​Emission (observation) matrix
d\mathbf{d}dDxD_xDx​Emission bias
R\mathbf{R}RDx×DxD_x \times D_xDx​×Dx​Emission noise covariance

Special cases. Setting A=0\mathbf{A} = \mathbf{0}A=0 and Q=I\mathbf{Q} = \mathbf{I}Q=I recovers Probabilistic PCA (Chapter 6): the latent states are i.i.d. Gaussians. The LDS adds temporal dependence through the dynamics matrix A\mathbf{A}A.

Stability. The dynamics are stable iff all eigenvalues of A\mathbf{A}A satisfy ∣λi∣<1|\lambda_i| < 1∣λi​∣<1. If any eigenvalue exceeds 1 in magnitude, the latent state diverges.

The Kalman Filter¶

The forward pass for the LDS is the Kalman filter, which computes the filtering distributions

p(zt∣x1:t)=N(zt∣μt∣t, Σt∣t)p(\mathbf{z}_t \mid \mathbf{x}_{1:t}) = \mathcal{N}(\mathbf{z}_t \mid \boldsymbol{\mu}_{t|t},\, \boldsymbol{\Sigma}_{t|t})p(zt​∣x1:t​)=N(zt​∣μt∣t​,Σt∣t​)
(5)

via two alternating steps. Assuming the predicted distribution p(zt∣x1:t−1)=N(μt∣t−1,Σt∣t−1)p(\mathbf{z}_t \mid \mathbf{x}_{1:t-1}) = \mathcal{N}(\boldsymbol{\mu}_{t|t-1}, \boldsymbol{\Sigma}_{t|t-1})p(zt​∣x1:t−1​)=N(μt∣t−1​,Σt∣t−1​) is in hand, we proceed as follows.

Update Step¶

Condition on the new observation xt\mathbf{x}_txt​. The joint distribution of (zt,xt)(\mathbf{z}_t, \mathbf{x}_t)(zt​,xt​) given x1:t−1\mathbf{x}_{1:t-1}x1:t−1​ is jointly Gaussian:

(ztxt)∣ x1:t−1∼N ⁣((μt∣t−1Cμt∣t−1+d),  (Σt∣t−1Σt∣t−1C⊤CΣt∣t−1CΣt∣t−1C⊤+R)).\begin{pmatrix}\mathbf{z}_t \\ \mathbf{x}_t\end{pmatrix} \Bigg|\, \mathbf{x}_{1:t-1} \sim \mathcal{N}\!\left( \begin{pmatrix}\boldsymbol{\mu}_{t|t-1} \\ \mathbf{C}\boldsymbol{\mu}_{t|t-1}+\mathbf{d}\end{pmatrix},\; \begin{pmatrix} \boldsymbol{\Sigma}_{t|t-1} & \boldsymbol{\Sigma}_{t|t-1}\mathbf{C}^\top \\ \mathbf{C}\boldsymbol{\Sigma}_{t|t-1} & \mathbf{C}\boldsymbol{\Sigma}_{t|t-1}\mathbf{C}^\top + \mathbf{R} \end{pmatrix} \right).(zt​xt​​)∣∣​x1:t−1​∼N((μt∣t−1​Cμt∣t−1​+d​),(Σt∣t−1​CΣt∣t−1​​Σt∣t−1​C⊤CΣt∣t−1​C⊤+R​)).
(6)

Applying the conditional Gaussian formula (Chapter 2), the posterior is:

μt∣t=μt∣t−1+Kt δt,Σt∣t=(I−KtC) Σt∣t−1\boldsymbol{\mu}_{t|t} = \boldsymbol{\mu}_{t|t-1} + \mathbf{K}_t\,\boldsymbol{\delta}_t, \qquad \boldsymbol{\Sigma}_{t|t} = (\mathbf{I} - \mathbf{K}_t\mathbf{C})\,\boldsymbol{\Sigma}_{t|t-1}μt∣t​=μt∣t−1​+Kt​δt​,Σt∣t​=(I−Kt​C)Σt∣t−1​
(7)

where

δt=xt−Cμt∣t−1−d(innovation)\boldsymbol{\delta}_t = \mathbf{x}_t - \mathbf{C}\boldsymbol{\mu}_{t|t-1} - \mathbf{d} \qquad\text{(innovation)}δt​=xt​−Cμt∣t−1​−d(innovation)
(8)
St=CΣt∣t−1C⊤+R(innovation covariance)\mathbf{S}_t = \mathbf{C}\boldsymbol{\Sigma}_{t|t-1}\mathbf{C}^\top + \mathbf{R} \qquad\text{(innovation covariance)}St​=CΣt∣t−1​C⊤+R(innovation covariance)
(9)
Kt=Σt∣t−1C⊤St−1(Kalman gain)\mathbf{K}_t = \boldsymbol{\Sigma}_{t|t-1}\mathbf{C}^\top\mathbf{S}_t^{-1} \qquad\text{(Kalman gain)}Kt​=Σt∣t−1​C⊤St−1​(Kalman gain)
(10)

The Kalman gain Kt\mathbf{K}_tKt​ interpolates between the prior prediction and the observation: when R→0\mathbf{R} \to 0R→0 (perfect sensors) Kt→C−1\mathbf{K}_t \to \mathbf{C}^{-1}Kt​→C−1 and we trust the data entirely; when R→∞\mathbf{R} \to \inftyR→∞ (useless sensors) Kt→0\mathbf{K}_t \to 0Kt​→0 and we ignore the data.

Predict Step¶

Marginalise over zt\mathbf{z}_tzt​ to obtain the predicted distribution for time t+1t+1t+1:

μt+1∣t=Aμt∣t+b,Σt+1∣t=AΣt∣tA⊤+Q.\boldsymbol{\mu}_{t+1|t} = \mathbf{A}\boldsymbol{\mu}_{t|t} + \mathbf{b}, \qquad \boldsymbol{\Sigma}_{t+1|t} = \mathbf{A}\boldsymbol{\Sigma}_{t|t}\mathbf{A}^\top + \mathbf{Q}.μt+1∣t​=Aμt∣t​+b,Σt+1∣t​=AΣt∣t​A⊤+Q.
(11)

Base case: μ1∣0=μ0\boldsymbol{\mu}_{1|0} = \boldsymbol{\mu}_0μ1∣0​=μ0​ and Σ1∣0=Σ0\boldsymbol{\Sigma}_{1|0} = \boldsymbol{\Sigma}_0Σ1∣0​=Σ0​.

Marginal Likelihood¶

As a byproduct, each update step gives the one-step predictive likelihood:

p(xt∣x1:t−1)=N(xt∣Cμt∣t−1+d,  St),p(\mathbf{x}_t \mid \mathbf{x}_{1:t-1}) = \mathcal{N}(\mathbf{x}_t \mid \mathbf{C}\boldsymbol{\mu}_{t|t-1}+\mathbf{d},\; \mathbf{S}_t),p(xt​∣x1:t−1​)=N(xt​∣Cμt∣t−1​+d,St​),
(12)

so log⁡p(x1:T)=∑t=1Tlog⁡p(xt∣x1:t−1)\log p(\mathbf{x}_{1:T}) = \sum_{t=1}^T \log p(\mathbf{x}_t \mid \mathbf{x}_{1:t-1})logp(x1:T​)=∑t=1T​logp(xt​∣x1:t−1​).

The RTS Smoother¶

The Kalman filter only uses past observations; it produces the filtering distribution p(zt∣x1:t)p(\mathbf{z}_t \mid \mathbf{x}_{1:t})p(zt​∣x1:t​). The smoothing distribution p(zt∣x1:T)p(\mathbf{z}_t \mid \mathbf{x}_{1:T})p(zt​∣x1:T​) uses all observations and is generally tighter.

Backward Recursion¶

By the Markov property, zt\mathbf{z}_tzt​ is conditionally independent of xt+1:T\mathbf{x}_{t+1:T}xt+1:T​ given zt+1\mathbf{z}_{t+1}zt+1​. This gives:

p(zt∣zt+1,x1:T)=p(zt∣zt+1,x1:t)=p(zt∣x1:t) p(zt+1∣zt)p(zt+1∣x1:t).p(\mathbf{z}_t \mid \mathbf{z}_{t+1}, \mathbf{x}_{1:T}) = p(\mathbf{z}_t \mid \mathbf{z}_{t+1}, \mathbf{x}_{1:t}) = \frac{p(\mathbf{z}_t \mid \mathbf{x}_{1:t})\, p(\mathbf{z}_{t+1} \mid \mathbf{z}_t)}{p(\mathbf{z}_{t+1} \mid \mathbf{x}_{1:t})}.p(zt​∣zt+1​,x1:T​)=p(zt​∣zt+1​,x1:t​)=p(zt+1​∣x1:t​)p(zt​∣x1:t​)p(zt+1​∣zt​)​.
(13)

Marginalising over zt+1\mathbf{z}_{t+1}zt+1​:

p(zt∣x1:T)=p(zt∣x1:t)∫p(zt+1∣zt) p(zt+1∣x1:T)p(zt+1∣x1:t) dzt+1.p(\mathbf{z}_t \mid \mathbf{x}_{1:T}) = p(\mathbf{z}_t \mid \mathbf{x}_{1:t}) \int \frac{p(\mathbf{z}_{t+1} \mid \mathbf{z}_t)\, p(\mathbf{z}_{t+1} \mid \mathbf{x}_{1:T})}{p(\mathbf{z}_{t+1} \mid \mathbf{x}_{1:t})} \, d\mathbf{z}_{t+1}.p(zt​∣x1:T​)=p(zt​∣x1:t​)∫p(zt+1​∣x1:t​)p(zt+1​∣zt​)p(zt+1​∣x1:T​)​dzt+1​.
(14)

For the Gaussian LDS all three factors are Gaussian and the integral is tractable. The result is the Rauch-Tung-Striebel (RTS) smoother:

p(zt∣x1:T)=N(μt∣T, Σt∣T)p(\mathbf{z}_t \mid \mathbf{x}_{1:T}) = \mathcal{N}(\boldsymbol{\mu}_{t|T},\, \boldsymbol{\Sigma}_{t|T})p(zt​∣x1:T​)=N(μt∣T​,Σt∣T​)
(15)

with backward recursion (starting from μT∣T\boldsymbol{\mu}_{T|T}μT∣T​, ΣT∣T\boldsymbol{\Sigma}_{T|T}ΣT∣T​ from the filter):

Gt=Σt∣tA⊤Σt+1∣t−1\mathbf{G}_t = \boldsymbol{\Sigma}_{t|t}\mathbf{A}^\top \boldsymbol{\Sigma}_{t+1|t}^{-1}Gt​=Σt∣t​A⊤Σt+1∣t−1​
(16)
μt∣T=μt∣t+Gt(μt+1∣T−μt+1∣t)\boldsymbol{\mu}_{t|T} = \boldsymbol{\mu}_{t|t} + \mathbf{G}_t(\boldsymbol{\mu}_{t+1|T} - \boldsymbol{\mu}_{t+1|t})μt∣T​=μt∣t​+Gt​(μt+1∣T​−μt+1∣t​)
(17)
Σt∣T=Σt∣t+Gt(Σt+1∣T−Σt+1∣t)Gt⊤\boldsymbol{\Sigma}_{t|T} = \boldsymbol{\Sigma}_{t|t} + \mathbf{G}_t(\boldsymbol{\Sigma}_{t+1|T} - \boldsymbol{\Sigma}_{t+1|t})\mathbf{G}_t^\topΣt∣T​=Σt∣t​+Gt​(Σt+1∣T​−Σt+1∣t​)Gt⊤​
(18)

The smoother gain Gt\mathbf{G}_tGt​ plays the same role as the Kalman gain: it determines how much the smoothed estimate at time ttt is adjusted based on new information from time t+1t+1t+1.

Information Form and Sparse Linear Algebra¶

There is an elegant alternative view. Write the joint posterior as:

p(z1:T∣x1:T)∝N(z1∣μ0,Σ0)∏t=2TN(zt∣Azt−1+b,Q)∏t=1TN(xt∣Czt+d,R).p(\mathbf{z}_{1:T} \mid \mathbf{x}_{1:T}) \propto \mathcal{N}(\mathbf{z}_1 \mid \boldsymbol{\mu}_0, \boldsymbol{\Sigma}_0) \prod_{t=2}^T \mathcal{N}(\mathbf{z}_t \mid \mathbf{A}\mathbf{z}_{t-1}+\mathbf{b}, \mathbf{Q}) \prod_{t=1}^T \mathcal{N}(\mathbf{x}_t \mid \mathbf{C}\mathbf{z}_t+\mathbf{d}, \mathbf{R}).p(z1:T​∣x1:T​)∝N(z1​∣μ0​,Σ0​)t=2∏T​N(zt​∣Azt−1​+b,Q)t=1∏T​N(xt​∣Czt​+d,R).
(19)

Expanding and collecting, the exponent is a quadratic form in z1:T∈RTDz\mathbf{z}_{1:T} \in \mathbb{R}^{T D_z}z1:T​∈RTDz​, so the posterior is a multivariate Gaussian with natural parameters J∈RTDz×TDz\mathbf{J} \in \mathbb{R}^{TD_z \times TD_z}J∈RTDz​×TDz​ (precision matrix) and h∈RTDz\mathbf{h} \in \mathbb{R}^{TD_z}h∈RTDz​.

The key structural observation is that J\mathbf{J}J is block-tridiagonal: only the diagonal blocks Jt,t\mathbf{J}_{t,t}Jt,t​ and the off-diagonal blocks Jt,t+1\mathbf{J}_{t,t+1}Jt,t+1​ are non-zero, reflecting the Markov structure. The posterior mean J−1h\mathbf{J}^{-1}\mathbf{h}J−1h can therefore be computed in O(TDz3)O(T D_z^3)O(TDz3​) rather than the naive O(T3Dz3)O(T^3 D_z^3)O(T3Dz3​) by exploiting the block-tridiagonal sparsity — and this block-tridiagonal solve is exactly equivalent to the Kalman filter and RTS smoother.

Message passing = sparse linear algebra

The forward-backward algorithm for a Gaussian LDS is equivalent to solving the block-tridiagonal linear system Jμ=h\mathbf{J}\boldsymbol{\mu} = \mathbf{h}Jμ=h with a block LDL decomposition. This duality extends to other graphical models: belief propagation on a tree is always equivalent to solving a sparse linear system.

Beyond the Linear Gaussian Case¶

For nonlinear dynamics zt=f(zt−1)+noise\mathbf{z}_t = f(\mathbf{z}_{t-1}) + \text{noise}zt​=f(zt−1​)+noise or non-Gaussian noise, the Kalman filter no longer applies exactly. Common approximations include:

MethodIdea
Extended Kalman Filter (EKF)Linearise fff around the current estimate
Unscented Kalman Filter (UKF)Propagate a set of sigma points through fff
Sequential Monte Carlo (SMC)Represent p(zt∣x1:t)p(\mathbf{z}_t \mid \mathbf{x}_{1:t})p(zt​∣x1:t​) by weighted particles

Sequential Monte Carlo. The forward messages are proportional to the predictive distributions p(zt∣x1:t−1)p(\mathbf{z}_t \mid \mathbf{x}_{1:t-1})p(zt​∣x1:t−1​). We can approximate this distribution with SSS weighted particles {(zt(s),wt(s))}\{(\mathbf{z}_t^{(s)}, w_t^{(s)})\}{(zt(s)​,wt(s)​)}:

  1. Sample zt(s)∼p(zt∣zt−1(s))\mathbf{z}_t^{(s)} \sim p(\mathbf{z}_t \mid \mathbf{z}_{t-1}^{(s)})zt(s)​∼p(zt​∣zt−1(s)​) (propagate particles).

  2. Weight wt(s)=p(xt∣zt(s))w_t^{(s)} = p(\mathbf{x}_t \mid \mathbf{z}_t^{(s)})wt(s)​=p(xt​∣zt(s)​) (score against observation).

  3. Normalise and resample with replacement according to wˉt(s)\bar{w}_t^{(s)}wˉt(s)​.

The marginal likelihood is approximated by log⁡p(x1:T)≈∑tlog⁡1S∑swt(s)\log p(\mathbf{x}_{1:T}) \approx \sum_t \log \frac{1}{S}\sum_s w_t^{(s)}logp(x1:T​)≈∑t​logS1​∑s​wt(s)​.

# ── Kalman filter ─────────────────────────────────────────────────────────────

def kalman_filter(y, A, b, Q, C, d, R, mu0, Sigma0):
    '''Kalman filter for a linear Gaussian SSM.

    Model:
        z_1      ~ N(mu0, Sigma0)
        z_t|z_{t-1} ~ N(A z_{t-1} + b, Q)
        x_t|z_t  ~ N(C z_t + d, R)

    Args:
        y:      (T, Dx)  observations
        A:      (Dz, Dz) dynamics matrix
        b:      (Dz,)    dynamics bias
        Q:      (Dz, Dz) dynamics noise covariance
        C:      (Dx, Dz) emission matrix
        d:      (Dx,)    emission bias
        R:      (Dx, Dx) emission noise covariance
        mu0:    (Dz,)    initial state mean
        Sigma0: (Dz, Dz) initial state covariance
    Returns:
        filtered_means:  (T, Dz)      mu_{t|t}
        filtered_covs:   (T, Dz, Dz)  Sigma_{t|t}
        predicted_means: (T, Dz)      mu_{t|t-1}
        predicted_covs:  (T, Dz, Dz)  Sigma_{t|t-1}
        log_marginal_likelihood: scalar
    '''
    T, Dx = y.shape
    Dz = mu0.shape[0]
    I = torch.eye(Dz)

    filtered_means  = torch.zeros(T, Dz)
    filtered_covs   = torch.zeros(T, Dz, Dz)
    predicted_means = torch.zeros(T, Dz)
    predicted_covs  = torch.zeros(T, Dz, Dz)
    log_ml = 0.0

    mu_pred    = mu0.clone()
    Sigma_pred = Sigma0.clone()

    for t in range(T):
        # Store predicted distribution p(z_t | x_{1:t-1})
        predicted_means[t] = mu_pred
        predicted_covs[t]  = Sigma_pred

        # ── Update: condition on x_t ──────────────────────────────────
        innov = y[t] - C @ mu_pred - d                   # innovation  (Dx,)
        S     = C @ Sigma_pred @ C.T + R                 # innov. cov  (Dx, Dx)
        K     = Sigma_pred @ C.T @ torch.linalg.inv(S)  # Kalman gain (Dz, Dx)

        mu_filt    = mu_pred + K @ innov
        Sigma_filt = (I - K @ C) @ Sigma_pred

        # Marginal likelihood contribution  log N(x_t | C mu_{t|t-1} + d, S)
        log_ml += dist.MultivariateNormal(C @ mu_pred + d, S).log_prob(y[t])

        filtered_means[t] = mu_filt
        filtered_covs[t]  = Sigma_filt

        # ── Predict: propagate to t+1 ─────────────────────────────────
        if t < T - 1:
            mu_pred    = A @ mu_filt + b
            Sigma_pred = A @ Sigma_filt @ A.T + Q

    return filtered_means, filtered_covs, predicted_means, predicted_covs, log_ml


# ── RTS Smoother ──────────────────────────────────────────────────────────────

def rts_smoother(A, filtered_means, filtered_covs, predicted_means, predicted_covs):
    '''Rauch-Tung-Striebel backward smoother.

    Args:
        A:               (Dz, Dz) dynamics matrix
        filtered_means:  (T, Dz)      from kalman_filter
        filtered_covs:   (T, Dz, Dz)
        predicted_means: (T, Dz)
        predicted_covs:  (T, Dz, Dz)
    Returns:
        smoothed_means: (T, Dz)      mu_{t|T}
        smoothed_covs:  (T, Dz, Dz)  Sigma_{t|T}
    '''
    T = filtered_means.shape[0]
    smoothed_means = filtered_means.clone()
    smoothed_covs  = filtered_covs.clone()

    for t in range(T - 2, -1, -1):
        # Smoother gain  G_t = Sigma_{t|t} A^T Sigma_{t+1|t}^{-1}
        G = filtered_covs[t] @ A.T @ torch.linalg.inv(predicted_covs[t + 1])
        smoothed_means[t] = (filtered_means[t]
                             + G @ (smoothed_means[t + 1] - predicted_means[t + 1]))
        smoothed_covs[t]  = (filtered_covs[t]
                             + G @ (smoothed_covs[t + 1] - predicted_covs[t + 1]) @ G.T)

    return smoothed_means, smoothed_covs

import numpy as np

dt = 0.4
T_demo = 60
torch.manual_seed(42)

# Model parameters
A = torch.eye(4)
A[0, 2] = dt
A[1, 3] = dt

b      = torch.zeros(4)
Q      = torch.diag(torch.tensor([1e-4, 1e-4, 0.05, 0.05]))

C      = torch.zeros(2, 4)
C[0, 0] = 1.0
C[1, 1] = 1.0
d      = torch.zeros(2)

mu0    = torch.tensor([0.0, 0.0, 0.8, 0.3])

# ── Simulate from the model ───────────────────────────────────────────────────

# ── Helper: draw a 2-sigma covariance ellipse (numpy-based) ──────────────────
def cov_ellipse(ax, mean_np, cov_np, n_std=2.0, **kwargs):
    from matplotlib.patches import Ellipse
    eigenvalues, eigenvectors = np.linalg.eigh(cov_np)
    angle = np.degrees(np.arctan2(eigenvectors[1, -1], eigenvectors[0, -1]))
    width, height = 2 * n_std * np.sqrt(eigenvalues)
    ell = Ellipse(xy=mean_np, width=width, height=height, angle=angle, **kwargs)
    ax.add_patch(ell)

The Kalman filter and RTS smoother are the exact continuous-state analogues of the discrete forward-backward algorithm: they exploit the Markov structure to compute marginals in linear time.

Learning. Given the smoothed marginals, the EM M-step for LDS parameters has closed forms analogous to the mixture model case. The sufficient statistics are E[zt]\mathbb{E}[\mathbf{z}_t]E[zt​], E[ztzt⊤]\mathbb{E}[\mathbf{z}_t \mathbf{z}_t^\top]E[zt​zt⊤​], and E[ztzt−1⊤]\mathbb{E}[\mathbf{z}_t \mathbf{z}_{t-1}^\top]E[zt​zt−1⊤​], all of which follow from the smoothed means and covariances. This is sometimes called the EM algorithm for LDS or the Shumway-Stoffer algorithm Shumway & Stoffer, 2000.
