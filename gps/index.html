Gaussian Processes

Prerequisites: This chapter builds on Multivariate Normal and Hierarchical Gaussian Models. Gaussian processes generalize multivariate Gaussian distributions to infinite-dimensional function spaces.

Reading: Williams & Rasmussen, 1996

A Gaussian process (GP) is a distribution over functions. Rather than placing a prior over a finite set of parameters, a GP places a prior over the function itself, giving a flexible non-parametric approach to regression and classification. This chapter covers:

  1. The GP definition and common kernels

  2. Exact GP regression

  3. GP classification via augmentation

  4. Elliptical slice sampling

  5. Sparse GPs and inducing points

Gaussian Processes¶

We say f∼GP(μ(⋅),K(⋅,⋅))f \sim \mathcal{GP}(\mu(\cdot), K(\cdot, \cdot))f∼GP(μ(⋅),K(⋅,⋅)) if for every finite set of inputs {x1,…,xN}⊂RD\{\mathbf{x}_1, \ldots, \mathbf{x}_N\} \subset \mathbb{R}^D{x1​,…,xN​}⊂RD, the joint distribution of the function values is Gaussian:

[f(x1)⋮f(xN)]∼N ⁣([μ(x1)⋮μ(xN)],  [K(x1,x1)⋯K(x1,xN)⋮⋮K(xN,x1)⋯K(xN,xN)]).\begin{bmatrix} f(\mathbf{x}_1) \\ \vdots \\ f(\mathbf{x}_N) \end{bmatrix} \sim \mathcal{N}\!\left( \begin{bmatrix} \mu(\mathbf{x}_1) \\ \vdots \\ \mu(\mathbf{x}_N) \end{bmatrix},\; \begin{bmatrix} K(\mathbf{x}_1, \mathbf{x}_1) & \cdots & K(\mathbf{x}_1, \mathbf{x}_N) \\ \vdots & & \vdots \\ K(\mathbf{x}_N, \mathbf{x}_1) & \cdots & K(\mathbf{x}_N, \mathbf{x}_N) \end{bmatrix} \right).⎣⎡​f(x1​)⋮f(xN​)​⎦⎤​∼N⎝⎛​⎣⎡​μ(x1​)⋮μ(xN​)​⎦⎤​,⎣⎡​K(x1​,x1​)⋮K(xN​,x1​)​⋯⋯​K(x1​,xN​)⋮K(xN​,xN​)​⎦⎤​⎠⎞​.
(1)

Here μ:RD→R\mu: \mathbb{R}^D \to \mathbb{R}μ:RD→R is the mean function and K:RD×RD→RK: \mathbb{R}^D \times \mathbb{R}^D \to \mathbb{R}K:RD×RD→R is the kernel (covariance function). The matrix of kernel evaluations is called the Gram matrix. The kernel must be positive definite: the Gram matrix must be PD for every finite set of inputs.

From Linear Regression to GPs¶

GPs are the natural limit of Bayesian linear regression with infinitely many features. Recall that with feature map ϕ(x)=(ϕ1(x),…,ϕP(x))⊤\boldsymbol{\phi}(x) = (\phi_1(x), \ldots, \phi_P(x))^\topϕ(x)=(ϕ1​(x),…,ϕP​(x))⊤ and Gaussian prior w∼N(0,λPI)\mathbf{w} \sim \mathcal{N}(\mathbf{0}, \frac{\lambda}{P}\mathbf{I})w∼N(0,Pλ​I), the function f(x)=ϕ(x)⊤wf(x) = \boldsymbol{\phi}(x)^\top \mathbf{w}f(x)=ϕ(x)⊤w is a GP with kernel

K(xi,xj)=λP ϕ(xi)⊤ϕ(xj).K(x_i, x_j) = \frac{\lambda}{P}\, \boldsymbol{\phi}(x_i)^\top \boldsymbol{\phi}(x_j).K(xi​,xj​)=Pλ​ϕ(xi​)⊤ϕ(xj​).
(2)

Example: squared exponential kernel. Use radial basis function features ϕp(x)=e−(x−cp)2/2ℓ2\phi_p(x) = e^{-(x - c_p)^2 / 2\ell^2}ϕp​(x)=e−(x−cp​)2/2ℓ2 centred at PPP equally-spaced points. Taking P→∞P \to \inftyP→∞:

lim⁡P→∞K(xi,xj)=λ∫−∞∞e−(xi−c)2/2ℓ2e−(xj−c)2/2ℓ2 dc=πℓλ  e−(xi−xj)2/4ℓ2.\lim_{P\to\infty} K(x_i, x_j) = \lambda \int_{-\infty}^\infty e^{-(x_i-c)^2/2\ell^2} e^{-(x_j-c)^2/2\ell^2}\,dc = \sqrt{\pi}\ell\lambda\; e^{-(x_i-x_j)^2/4\ell^2}.P→∞lim​K(xi​,xj​)=λ∫−∞∞​e−(xi​−c)2/2ℓ2e−(xj​−c)2/2ℓ2dc=π​ℓλe−(xi​−xj​)2/4ℓ2.
(3)

This is the squared exponential (SE) kernel (also called the RBF or Gaussian kernel):

KSE(xi,xj)=σ2exp⁡ ⁣(−(xi−xj)22ℓ2).K_{\mathrm{SE}}(x_i, x_j) = \sigma^2 \exp\!\left(-\frac{(x_i-x_j)^2}{2\ell^2}\right).KSE​(xi​,xj​)=σ2exp(−2ℓ2(xi​−xj​)2​).
(4)

The SE kernel corresponds to functions that are infinitely differentiable. The length-scale ℓ\ellℓ controls how quickly the function varies; σ2\sigma^2σ2 is the marginal variance.

Stationary Kernels and Bochner’s Theorem¶

Both SE and Matérn kernels are stationary: K(xi,xj)K(\mathbf{x}_i, \mathbf{x}_j)K(xi​,xj​) depends only on the difference Δ=xi−xj\boldsymbol{\Delta} = \mathbf{x}_i - \mathbf{x}_jΔ=xi​−xj​.

Bochner’s Theorem

A continuous function K(Δ)K(\boldsymbol{\Delta})K(Δ) is a valid stationary kernel iff it is the Fourier transform of a non-negative measure μ\muμ:

K(Δ)=∫e2πis⋅Δ dμ(s).K(\boldsymbol{\Delta}) = \int e^{2\pi i \mathbf{s}\cdot\boldsymbol{\Delta}}\,d\mu(\mathbf{s}).K(Δ)=∫e2πis⋅Δdμ(s).
(6)

If μ\muμ has density S(s)S(\mathbf{s})S(s) (the spectral density or power spectrum), then KKK and SSS are Fourier duals. (Theorem 4.1 of Williams & Rasmussen, 1996.)

Posterior over Function Values¶

Given observations (xn,yn)(\mathbf{x}_n, y_n)(xn​,yn​) with yn=f(xn)+ϵny_n = f(\mathbf{x}_n) + \epsilon_nyn​=f(xn​)+ϵn​, ϵn∼N(0,σ2)\epsilon_n \sim \mathcal{N}(0, \sigma^2)ϵn​∼N(0,σ2), the posterior over the NNN-vector f=[f(x1),…,f(xN)]⊤\mathbf{f} = [f(\mathbf{x}_1),\ldots,f(\mathbf{x}_N)]^\topf=[f(x1​),…,f(xN​)]⊤ is Gaussian:

p(f∣y)=N(f∣μ′,K′)p(\mathbf{f} \mid \mathbf{y}) = \mathcal{N}(\mathbf{f} \mid \boldsymbol{\mu}', \mathbf{K}')p(f∣y)=N(f∣μ′,K′)
(7)
K′=(K−1+σ−2I)−1,μ′=K′(K−1μ+σ−2y).\mathbf{K}' = \bigl(\mathbf{K}^{-1} + \sigma^{-2}\mathbf{I}\bigr)^{-1}, \qquad \boldsymbol{\mu}' = \mathbf{K}'\bigl(\mathbf{K}^{-1}\boldsymbol{\mu} + \sigma^{-2}\mathbf{y}\bigr).K′=(K−1+σ−2I)−1,μ′=K′(K−1μ+σ−2y).

p(y)=N(y∣μ, K+σ2I).p(\mathbf{y}) = \mathcal{N}(\mathbf{y} \mid \boldsymbol{\mu},\, \mathbf{K} + \sigma^2\mathbf{I}).p(y)=N(y∣μ,K+σ2I).

This can be used to select hyperparameters (kernel parameters, noise variance) by maximising log⁡p(y)\log p(\mathbf{y})logp(y).

Elliptical Slice Sampling¶

For non-conjugate GP models (e.g. GP classification with a logistic link), standard Gibbs can mix poorly because the GP prior induces strong correlations between function values.

Murray et al., 2010 proposed elliptical slice sampling (ESS), which exploits the Gaussian structure of the prior. The key observation is that any draw from N(0,Σ)\mathcal{N}(\mathbf{0}, \boldsymbol{\Sigma})N(0,Σ) can be written as

f=ν0sin⁡θ+ν1cos⁡θ,ν0,ν1∼N(0,Σ).\mathbf{f} = \boldsymbol{\nu}_0 \sin\theta + \boldsymbol{\nu}_1 \cos\theta, \qquad \boldsymbol{\nu}_0, \boldsymbol{\nu}_1 \sim \mathcal{N}(\mathbf{0}, \boldsymbol{\Sigma}).f=ν0​sinθ+ν1​cosθ,ν0​,ν1​∼N(0,Σ).
(17)

Given the current f\mathbf{f}f, a new proposal is made by rotating on the ellipse defined by f\mathbf{f}f and a fresh auxiliary draw ν\boldsymbol{\nu}ν:

f′=fcos⁡θ+νsin⁡θ,ν∼N(0,Σ).\mathbf{f}' = \mathbf{f}\cos\theta + \boldsymbol{\nu}\sin\theta, \qquad \boldsymbol{\nu} \sim \mathcal{N}(\mathbf{0}, \boldsymbol{\Sigma}).f′=fcosθ+νsinθ,ν∼N(0,Σ).
(18)

The angle θ\thetaθ is sampled by a slice over [0,2π)[0, 2\pi)[0,2π): start with a random bracket, evaluate the likelihood at the proposal, shrink the bracket if the proposal is rejected. The algorithm is always accepted eventually and leaves the posterior invariant.

Sparse GPs and Inducing Points¶

Exact GP inference costs O(N3)O(N^3)O(N3). For large NNN we need approximations. Inducing point methods Titsias, 2009 introduce M≪NM \ll NM≪N pseudo-inputs {zm}\{\mathbf{z}_m\}{zm​} and approximate the full posterior with one conditioned on the function values u=[f(z1),…,f(zM)]⊤\mathbf{u} = [f(\mathbf{z}_1),\ldots,f(\mathbf{z}_M)]^\topu=[f(z1​),…,f(zM​)]⊤.

Using the GP predictive distribution:

p(f∣u)=N(f∣KNMKMM−1u,  K~)K~=KNN−KNMKMM−1KMN.p(\mathbf{f} \mid \mathbf{u}) = \mathcal{N}(\mathbf{f} \mid \mathbf{K}_{NM}\mathbf{K}_{MM}^{-1}\mathbf{u},\; \tilde{\mathbf{K}}) \qquad \tilde{\mathbf{K}} = \mathbf{K}_{NN} - \mathbf{K}_{NM}\mathbf{K}_{MM}^{-1}\mathbf{K}_{MN}.p(f∣u)=N(f∣KNM​KMM−1​u,K~)K~=KNN​−KNM​KMM−1​KMN​.
(19)

Variational Bound¶

Titsias, 2009 lower-bounds log⁡p(y)\log p(\mathbf{y})logp(y) by:

L2=log⁡N(y∣0,  KNMKMM−1KMN+σ2I)−12σ2tr(K~).\mathcal{L}_2 = \log \mathcal{N}(\mathbf{y} \mid \mathbf{0},\; \mathbf{K}_{NM}\mathbf{K}_{MM}^{-1}\mathbf{K}_{MN} + \sigma^2\mathbf{I}) - \frac{1}{2\sigma^2}\mathrm{tr}(\tilde{\mathbf{K}}).L2​=logN(y∣0,KNM​KMM−1​KMN​+σ2I)−2σ21​tr(K~).
(20)

The first term is the likelihood of a low-rank GP; the trace term penalises unexplained prior variance. Computing L2\mathcal{L}_2L2​ costs O(NM2+M3)O(NM^2 + M^3)O(NM2+M3).

Stochastic Variational Inference¶

Hensman et al., 2013 showed that if we explicitly maintain a variational distribution q(u)=N(m,S)q(\mathbf{u}) = \mathcal{N}(\mathbf{m}, \mathbf{S})q(u)=N(m,S) rather than marginalising out u\mathbf{u}u, the ELBO factorises over data points:

L3=∑n=1NEq(f(xn))[log⁡p(yn∣f(xn))]−KL(q(u)∥p(u)).\mathcal{L}_3 = \sum_{n=1}^N \mathbb{E}_{q(f(\mathbf{x}_n))}[\log p(y_n \mid f(\mathbf{x}_n))] - \mathrm{KL}(q(\mathbf{u}) \| p(\mathbf{u})).L3​=n=1∑N​Eq(f(xn​))​[logp(yn​∣f(xn​))]−KL(q(u)∥p(u)).
(21)

This enables mini-batch stochastic gradient ascent, scaling GPs to millions of observations at O(M3)O(M^3)O(M3) per step.

def rbf_kernel(x1, x2, length_scale=1.0, variance=1.0):
    '''Squared exponential (RBF) kernel  k(x1, x2) = variance * exp(-r^2 / 2).

    Args:
        x1: (N1,)  1-D inputs
        x2: (N2,)  1-D inputs
    Returns:
        (N1, N2) Gram matrix
    '''
    r2 = ((x1[:, None] - x2[None, :]) / length_scale) ** 2
    return variance * torch.exp(-0.5 * r2)


def matern_kernel(x1, x2, length_scale=1.0, variance=1.0, nu=1.5):
    '''Matern kernel for nu in {0.5, 1.5, 2.5}.

    Args:
        x1: (N1,)  1-D inputs
        x2: (N2,)  1-D inputs
        nu: smoothness parameter (0.5, 1.5, or 2.5)
    Returns:
        (N1, N2) Gram matrix
    '''
    r = (x1[:, None] - x2[None, :]).abs() / length_scale
    if nu == 0.5:
        return variance * torch.exp(-r)
    elif nu == 1.5:
        sr = math.sqrt(3) * r
        return variance * (1 + sr) * torch.exp(-sr)
    elif nu == 2.5:
        sr = math.sqrt(5) * r
        return variance * (1 + sr + sr**2 / 3) * torch.exp(-sr)
    else:
        raise ValueError(f'Unsupported nu={nu}; use 0.5, 1.5, or 2.5')


# ── Exact GP regression ───────────────────────────────────────────────────────

def gp_posterior_predictive(x_train, y_train, x_test, kernel_fn,
                             noise_var=0.1, mu=0.0):
    '''Exact GP regression: posterior predictive mean and std.

    Model:
        f ~ GP(mu, K)
        y_n | f ~ N(f(x_n), noise_var)

    Args:
        x_train:    (N,)   training inputs
        y_train:    (N,)   training targets
        x_test:     (M,)   test inputs
        kernel_fn:  callable(x1, x2) -> (N1, N2) kernel matrix
        noise_var:  observation noise variance
        mu:         prior mean (scalar)
    Returns:
        mean_pred:  (M,)   posterior predictive mean
        std_pred:   (M,)   posterior predictive std
        log_ml:     scalar log marginal likelihood
    '''
    N = len(x_train)
    jitter = 1e-4

    K_nn = kernel_fn(x_train, x_train) + (noise_var + jitter) * torch.eye(N)  # jitter=1e-4
    k_ns = kernel_fn(x_train, x_test)   # (N, M)
    K_ss = kernel_fn(x_test,  x_test)   # (M, M)

    L     = torch.linalg.cholesky(K_nn)                        # (N, N)
    alpha = torch.cholesky_solve((y_train - mu).unsqueeze(1), L).squeeze(1)  # (N,)

    mean_pred = mu + k_ns.T @ alpha                             # (M,)

    v         = torch.linalg.solve_triangular(L, k_ns, upper=False)  # (N, M)
    var_pred   = K_ss.diagonal() - (v * v).sum(0)              # (M,)
    var_pred   = var_pred.clamp(min=0) + noise_var
    std_pred   = var_pred.sqrt()

    # Log marginal likelihood: log N(y | mu*1, K_nn)
    log_ml = (-0.5 * (y_train - mu) @ alpha
              - L.diagonal().log().sum()
              - 0.5 * N * math.log(2 * math.pi))

    return mean_pred, std_pred, log_ml

# ── Helper: sample prior paths ────────────────────────────────────────────────
def sample_prior(x, kernel_fn, n_samples=5, seed=0):
    torch.manual_seed(seed)
    K = kernel_fn(x, x) + 1e-4 * torch.eye(len(x))
    L = torch.linalg.cholesky(K)
    eps = torch.randn(len(x), n_samples)
    return L @ eps   # (N, n_samples)
