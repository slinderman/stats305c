Deep SSMs and Linear Attention

Prerequisites

This chapter builds on Recurrent Neural Networks, Linear Dynamical Systems, and Transformers.

Two tensions define modern sequence modeling. On the training side, recurrent models like RNNs process tokens one at a time, leaving GPU parallelism underutilized on long sequences. On the inference side, Transformers compute all pairwise token interactions, incurring O(N2)O(N^2)O(N2) time and memory that becomes prohibitive for long contexts. Can we get the best of both worlds — parallel training and efficient inference?

This chapter shows that the answer is yes, for a broad family of models that can be written as linear recurrences. A linear recurrence is simultaneously a convolution (fast parallel training via FFT or parallel scan) and a fixed-size hidden-state update (fast O(1)O(1)O(1)-memory inference). Structured SSMs exploit this duality with carefully designed dynamics matrices; linear attention recovers it by approximating the softmax kernel.

Deep State Space Models¶

Linear Recurrent Neural Networks¶

A linear RNN evolves a hidden state ht∈Rp\mbh_t \in \reals^pht​∈Rp via

ht=Aht−1+Bxt,yt=Cht+d,\mbh_t = \mbA \mbh_{t-1} + \mbB \mbx_t, \qquad \mby_t = \mbC \mbh_t + \mbd,ht​=Aht−1​+Bxt​,yt​=Cht​+d,
(1)

with h0=0\mbh_0 = \mathbf{0}h0​=0. Because linear operators compose, unrolling the recurrence gives

yt=C∑s=0t−1AsBxt−s+d=[K⊛x]t,\mby_t = \mbC \sum_{s=0}^{t-1} \mbA^s \mbB \mbx_{t-s} + \mbd = [\mbK \circledast \mbx]_t,yt​=Cs=0∑t−1​AsBxt−s​+d=[K⊛x]t​,
(2)

where the SSM kernel is K=[CB    CAB    CA2B    ⋯ ]\mbK = [\mbC\mbB \;\; \mbC\mbA\mbB \;\; \mbC\mbA^2\mbB \;\; \cdots]K=[CBCABCA2B⋯].

This recurrence–convolution duality is the key computational property:

  • Recurrent form: O(Np)O(Np)O(Np) inference, O(p)O(p)O(p) memory — ideal for generation.

  • Convolutional form: evaluate the full kernel once and convolve via FFT, O(Nlog⁡N)O(N \log N)O(NlogN) — ideal for training on long sequences.

When the dynamics At\mbA_tAt​ are input-dependent (time-varying), the fixed kernel no longer applies. Training then requires a parallel scan over the sequence — an O(log⁡T)O(\log T)O(logT)-depth divide-and-conquer algorithm that exploits the associativity of affine function composition. See Parallelizing Nonlinear RNNs for a detailed treatment of the parallel scan and its extensions to nonlinear recurrences.

S4: A Continuous-Time Deep SSM¶

S4 Gu et al., 2022 is a deep SSM that derives its discrete-time recurrence from a continuous-time dynamical system, giving a principled way to initialize A\mbAA so that the model captures long-range dependencies. A linear time-invariant (LTI) system evolves a hidden state h(t)∈Rph(t) \in \mathbb{R}^ph(t)∈Rp via:

h′(t)=A h(t)+B x(t),y(t)=C⊤h(t),h'(t) = A\, h(t) + B\, x(t), \qquad y(t) = C^\top h(t),h′(t)=Ah(t)+Bx(t),y(t)=C⊤h(t),
(3)

Discretizing with step size Δ\DeltaΔ (zero-order hold) yields the recurrent form:

ht=Aˉ ht−1+Bˉ xt,yt=C⊤ht,h_t = \bar{A}\, h_{t-1} + \bar{B}\, x_t, \qquad y_t = C^\top h_t,ht​=Aˉht−1​+Bˉxt​,yt​=C⊤ht​,
(4)

where Aˉ=eΔA\bar{A} = e^{\Delta A}Aˉ=eΔA and Bˉ=(Aˉ−I)A−1B\bar{B} = (\bar{A} - I) A^{-1} BBˉ=(Aˉ−I)A−1B. Because Aˉ\bar{A}Aˉ is fixed, the output is a convolution with the SSM kernel Kˉ=(C⊤Bˉ,  C⊤AˉBˉ,  C⊤Aˉ2Bˉ,…)\bar{K} = (C^\top \bar{B},\; C^\top \bar{A} \bar{B},\; C^\top \bar{A}^2 \bar{B}, \ldots)Kˉ=(C⊤Bˉ,C⊤AˉBˉ,C⊤Aˉ2Bˉ,…):

yt=(Kˉ∗x)t.y_t = (\bar{K} * x)_t.yt​=(Kˉ∗x)t​.
(5)

The remaining challenge is computing Kˉ\bar{K}Kˉ efficiently. Computing eΔAe^{\Delta A}eΔA naively is O(p3)O(p^3)O(p3). S4 addresses this by initializing AAA using the HiPPO framework Gu et al., 2020, which constructs AAA as a structured matrix designed to optimally memorize the input history via polynomial projections. The HiPPO-LegS matrix has entries:

Ank={−(2n+1)1/2(2k+1)1/2n>k−(n+1)n=k0n<kA_{nk} = \begin{cases} -(2n+1)^{1/2}(2k+1)^{1/2} & n > k \\ -(n+1) & n = k \\ 0 & n < k \end{cases}Ank​=⎩⎨⎧​−(2n+1)1/2(2k+1)1/2−(n+1)0​n>kn=kn<k​
(6)

S4 exploits the fact that HiPPO-LegS is a normal plus low-rank (NPLR) matrix to compute the SSM kernel in O(p+Nlog⁡N)O(p + N \log N)O(p+NlogN) time via the fast Fourier transform.

S5: Diagonal SSMs with Parallel Scans¶

S5S5S5 Smith et al., 2023 simplifies S4 by diagonalizing the state matrix: Λ=P−1AP\Lambda = P^{-1} A PΛ=P−1AP where Λ\LambdaΛ is diagonal. With h~t=P−1ht\tilde{h}_t = P^{-1} h_th~t​=P−1ht​, the recurrence decouples into ppp independent scalar recurrences:

h~t,i=λih~t−1,i+B~ixt,yt=Re(C⊤h~t).\tilde{h}_{t,i} = \lambda_i \tilde{h}_{t-1,i} + \tilde{B}_i x_t, \qquad y_t = \mathrm{Re}(C^\top \tilde{h}_t).h~t,i​=λi​h~t−1,i​+B~i​xt​,yt​=Re(C⊤h~t​).
(7)

Each mode decays independently at rate ∣λi∣|\lambda_i|∣λi​∣; stability requires ∣λi∣<1|\lambda_i| < 1∣λi​∣<1. S5 computes all NNN hidden states in parallel using an associative parallel scan, reducing training time from O(Np)O(Np)O(Np) sequential steps to O(plog⁡N)O(p \log N)O(plogN) parallel steps.

Mamba: Selective SSMs¶

A fundamental limitation of LTI SSMs is that the transition matrices Aˉ\bar{A}Aˉ and Bˉ\bar{B}Bˉ are input-independent: every token is processed identically regardless of content. This prevents the model from selectively retaining or discarding information based on context.

Mamba Gu & Dao, 2023 introduces a selection mechanism by making BtB_tBt​, CtC_tCt​, and the discretization step Δt\Delta_tΔt​ functions of the input xtx_txt​:

ht=Aˉt ht−1+Bˉt xt,yt=Ct⊤ht,h_t = \bar{A}_t\, h_{t-1} + \bar{B}_t\, x_t, \qquad y_t = C_t^\top h_t,ht​=Aˉt​ht−1​+Bˉt​xt​,yt​=Ct⊤​ht​,
(8)

where Aˉt=eΔtA\bar{A}_t = e^{\Delta_t A}Aˉt​=eΔt​A, Bˉt=(Aˉt−I)A−1Bt\bar{B}_t = (\bar{A}_t - I) A^{-1} B_tBˉt​=(Aˉt​−I)A−1Bt​, and Δt,Bt,Ct\Delta_t, B_t, C_tΔt​,Bt​,Ct​ are computed from xtx_txt​ via small linear projections.

Because parameters now vary with ttt, the convolutional view no longer applies — the system is time-varying, and yty_tyt​ depends on all of x1,…,xtx_1, \ldots, x_tx1​,…,xt​ in a non-linear way. Training requires a selective scan: a hardware-aware parallel algorithm that exploits the structure of the recurrence without materializing intermediate states in memory.

The selection mechanism gives Mamba attention-like content-dependent routing while preserving the O(Np)O(N p)O(Np) cost of a recurrent model.

Linear Attention¶

The link between deep SSMs and Transformers (see the Transformers chapter) may be closer than it appears. Both Mamba and softmax attention compute content-dependent outputs — Mamba via input-driven gates, attention via pairwise dot products. The key difference is architectural: SSMs maintain a fixed-size hidden state updated recurrently, while attention computes all pairwise interactions simultaneously, with no fixed-size bottleneck but at O(N2)O(N^2)O(N2) cost. Linearizing the attention kernel bridges the two, recovering a recurrent model from attention. Given queries Q∈RN×dQ \in \mathbb{R}^{N \times d}Q∈RN×d, keys K∈RN×dK \in \mathbb{R}^{N \times d}K∈RN×d, and values V∈RN×dV \in \mathbb{R}^{N \times d}V∈RN×d, the output of causal (autoregressive) softmax attention at position iii is:

yi=∑j=1iexp⁡ ⁣(qi⊤kj/d)vj∑j=1iexp⁡ ⁣(qi⊤kj/d).y_i = \frac{\sum_{j=1}^{i} \exp\!\left(q_i^\top k_j / \sqrt{d}\right) v_j}{\sum_{j=1}^{i} \exp\!\left(q_i^\top k_j / \sqrt{d}\right)}.yi​=∑j=1i​exp(qi⊤​kj​/d​)∑j=1i​exp(qi⊤​kj​/d​)vj​​.
(9)

In matrix form (with the causal mask Mij=I[j≤i]M_{ij} = \mathbb{I}[j \leq i]Mij​=I[j≤i]):

Y=softmax ⁣(QK⊤d+log⁡M)V,Y = \mathrm{softmax}\!\left(\frac{QK^\top}{\sqrt{d}} + \log M\right) V,Y=softmax(d​QK⊤​+logM)V,
(10)

where log⁡M\log MlogM sets future entries to −∞-\infty−∞. Computing this requires materializing the N×NN \times NN×N attention matrix, costing O(N2d)O(N^2 d)O(N2d) time and O(N2)O(N^2)O(N2) memory.

Katharopoulos et al. (2020) observe that the softmax is a kernel function, exp⁡(q⊤k/d)=κ(q,k)\exp(q^\top k / \sqrt{d}) = \kappa(q, k)exp(q⊤k/d​)=κ(q,k), and that replacing it with a kernel with explicit feature maps, κ(q,k)=ϕ(q)⊤ϕ(k)\kappa(q, k) = \phi(q)^\top \phi(k)κ(q,k)=ϕ(q)⊤ϕ(k), unlocks a dramatic simplification. The causal attention output becomes:

yi=ϕ(qi)⊤∑j≤iϕ(kj)vj⊤ϕ(qi)⊤∑j≤iϕ(kj).y_i = \frac{\phi(q_i)^\top \sum_{j \leq i} \phi(k_j) v_j^\top}{\phi(q_i)^\top \sum_{j \leq i} \phi(k_j)}.yi​=ϕ(qi​)⊤∑j≤i​ϕ(kj​)ϕ(qi​)⊤∑j≤i​ϕ(kj​)vj⊤​​.
(11)

The key is that ϕ(qi)\phi(q_i)ϕ(qi​) acts on the accumulated outer products, which can be built up incrementally as a linear recurrence. Defining the hidden state and normalizer:

St=St−1+ϕ(kt)vt⊤∈Rr×d,zt=zt−1+ϕ(kt)∈Rr,S_t = S_{t-1} + \phi(k_t) v_t^\top \in \mathbb{R}^{r \times d}, \qquad z_t = z_{t-1} + \phi(k_t) \in \mathbb{R}^r,St​=St−1​+ϕ(kt​)vt⊤​∈Rr×d,zt​=zt−1​+ϕ(kt​)∈Rr,
(12)

the output at each step is yt=St⊤ϕ(qt)/(zt⊤ϕ(qt))y_t = S_t^\top \phi(q_t) / (z_t^\top \phi(q_t))yt​=St⊤​ϕ(qt​)/(zt⊤​ϕ(qt​)). This is a linear RNN with hidden state StS_tSt​ — O(Nrd)O(Nrd)O(Nrd) time and O(rd)O(rd)O(rd) memory, with no attention matrix. A simple feature map that keeps values positive is ϕ(x)=elu(x)+1\phi(x) = \mathrm{elu}(x) + 1ϕ(x)=elu(x)+1 elementwise, giving r=dr = dr=d.

Test-Time Regression¶

Linear attention can be reread as solving a regression problem online. At each step, the model has observed a sequence of key-value pairs (k1,v1),…,(kt,vt)(k_1, v_1), \ldots, (k_t, v_t)(k1​,v1​),…,(kt​,vt​), which form a training set, and must predict a value at a new query qtq_tqt​, the test point. The hidden state StS_tSt​ is the current regression solution, updated incrementally as new pairs arrive.

Concretely, consider fitting a linear map W:Rd→RdW : \reals^d \to \reals^dW:Rd→Rd to the accumulated pairs via ridge regression:

Wt=arg⁡min⁡W∑s≤t∥Wks−vs∥2+λ∥W∥F2.W_t = \arg\min_W \sum_{s \leq t} \|W k_s - v_s\|^2 + \lambda \|W\|_F^2.Wt​=argWmin​s≤t∑​∥Wks​−vs​∥2+λ∥W∥F2​.
(13)

The closed-form solution is Wt=(∑s≤tvsks⊤)(∑s≤tksks⊤+λI)−1W_t = \left(\sum_{s \leq t} v_s k_s^\top\right) \left(\sum_{s \leq t} k_s k_s^\top + \lambda I\right)^{-1}Wt​=(∑s≤t​vs​ks⊤​)(∑s≤t​ks​ks⊤​+λI)−1, and the prediction at qtq_tqt​ is yt=Wtqty_t = W_t q_tyt​=Wt​qt​ — kernel ridge regression with a linear kernel. The models above are all approximations to this solution:

  • Linear attention computes StqtS_t q_tSt​qt​ with St=∑s≤tϕ(ks)vs⊤S_t = \sum_{s \leq t} \phi(k_s) v_s^\topSt​=∑s≤t​ϕ(ks​)vs⊤​ — the numerator of the ridge regression estimator, omitting the Gram matrix inverse.

  • Softmax attention corresponds to kernel regression with the softmax kernel.

The remaining question is how to approximate the full ridge regression solution cheaply and online.

DeltaNet: Gradient Descent for Linear Regression¶

Rather than accumulating all key-value pairs, DeltaNet Yang et al., 2024 maintains a weight matrix StS_tSt​ and updates it by taking one step of gradient descent on the per-token least-squares loss ℓt(S)=12∥Skt−vt∥2\ell_t(S) = \tfrac{1}{2}\|S k_t - v_t\|^2ℓt​(S)=21​∥Skt​−vt​∥2. With step size βt\beta_tβt​, the gradient step gives the delta rule:

St=St−1−βt(St−1kt−vt)kt⊤=(I−βtktkt⊤) St−1+βtvtkt⊤.S_t = S_{t-1} - \beta_t (S_{t-1} k_t - v_t) k_t^\top = (I - \beta_t k_t k_t^\top)\, S_{t-1} + \beta_t v_t k_t^\top.St​=St−1​−βt​(St−1​kt​−vt​)kt⊤​=(I−βt​kt​kt⊤​)St−1​+βt​vt​kt⊤​.
(14)

This is a rank-1 correction: the old memory’s prediction St−1ktS_{t-1} k_tSt−1​kt​ is erased and the new target vtv_tvt​ is written in, both weighted by ktk_tkt​. Unlike linear attention (St=St−1+ϕ(kt)vt⊤S_t = S_{t-1} + \phi(k_t) v_t^\topSt​=St−1​+ϕ(kt​)vt⊤​, no forgetting), DeltaNet selectively overwrites memory associated with ktk_tkt​. Output is yt=St⊤qty_t = S_t^\top q_tyt​=St⊤​qt​.

With keys and queries ℓ2\ell_2ℓ2​-normalized and βt\beta_tβt​ a learned sigmoid gate, DeltaNet is a gated linear RNN with content-dependent forgetting — more expressive than linear attention, cheaper than softmax attention (O(Nd2)O(Nd^2)O(Nd2) vs. O(N2d)O(N^2 d)O(N2d)). Yang et al. (2024) derive an efficient parallel training algorithm based on the WY representation of Householder products.

Test-Time Training¶

Test-time training (TTT) Yu et al., 2024 takes the regression perspective to its logical conclusion: the hidden state WtW_tWt​ is the weights of a small nonlinear model fWt:Rd→Rdf_{W_t} : \reals^d \to \reals^dfWt​​:Rd→Rd (e.g., a two-layer MLP), updated online by gradient descent on the reconstruction loss:

ℓt(W)=12∥fW(kt)−vt∥2.\ell_t(W) = \tfrac{1}{2}\|f_W(k_t) - v_t\|^2.ℓt​(W)=21​∥fW​(kt​)−vt​∥2.
(15)

One gradient step gives Wt=Wt−1−η∇Wℓt(Wt−1)W_t = W_{t-1} - \eta \nabla_W \ell_t(W_{t-1})Wt​=Wt−1​−η∇W​ℓt​(Wt−1​), and output is yt=fWt(qt)y_t = f_{W_t}(q_t)yt​=fWt​​(qt​). When fWf_WfW​ is the linear map fW(x)=Wxf_W(x) = WxfW​(x)=Wx, this reduces exactly to DeltaNet with βt=η\beta_t = \etaβt​=η — so DeltaNet is TTT with the simplest possible hidden model. For nonlinear fWf_WfW​, TTT can take multiple gradient steps on mini-batches of context tokens, creating a meta-learned inner loop that trades compute for expressivity of the hidden state.
