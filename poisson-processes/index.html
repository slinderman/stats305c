Poisson Processes

Prerequisites

This chapter is largely self-contained. Familiarity with Bayesian Modeling and basic probability is assumed.

Poisson processes are fundamental stochastic models for random sets of points in time or space. In this chapter we study their defining properties, four ways to sample them, and how to go beyond simple Poisson models with self-exciting and doubly stochastic processes.

Defining Properties of a Poisson Process¶

A Poisson process is a stochastic process that generates a random set of points {xn}n=1N⊂X\{\mathbf{x}_n\}_{n=1}^N \subset \mathcal{X}{xn​}n=1N​⊂X. It is governed by an intensity function λ(x):X→R+\lambda(\mathbf{x}): \mathcal{X} \to \mathbb{R}_+λ(x):X→R+​.

Two key properties define a Poisson process:

  1. Property 1 (Poisson counts): The number of points in any measurable set A\mathcal{A}A follows a Poisson distribution,

    N(A)∼Poisson ⁣(∫Aλ(x) dx).N(\mathcal{A}) \sim \mathrm{Poisson}\!\left(\int_{\mathcal{A}} \lambda(\mathbf{x})\, d\mathbf{x} \right).N(A)∼Poisson(∫A​λ(x)dx).
    (1)

  2. Property 2 (Independence): Counts on disjoint sets are independent,

    N(A)⊥N(B)  ⟺  A∩B=∅.N(\mathcal{A}) \perp N(\mathcal{B}) \iff \mathcal{A} \cap \mathcal{B} = \varnothing.N(A)⊥N(B)⟺A∩B=∅.
    (2)

Common applications include modeling neural spike trains, locations of trees in a forest, arrival times of customers or web requests, and positions of photons on a detector.

Poisson Process Likelihood¶

From the top-down sampling procedure we can derive the likelihood. Given points {xn}n=1N⊂[0,T]\{x_n\}_{n=1}^N \subset [0, T]{xn​}n=1N​⊂[0,T]:

p ⁣({xn}n=1N∣λ)=e−∫0Tλ(x) dx∏n=1Nλ(xn).p\!\left(\{x_n\}_{n=1}^N \mid \lambda\right) = e^{-\int_0^T \lambda(x)\, dx} \prod_{n=1}^N \lambda(x_n).p({xn​}n=1N​∣λ)=e−∫0T​λ(x)dxn=1∏N​λ(xn​).
(3)

This can be understood as: the probability that no point falls outside the observed set (the exponential factor) times the probability density at each observed location (the product term).

Four Ways to Sample a Poisson Process¶

1. Top-Down Sampling¶

For a Poisson process with intensity λ(x)\lambda(\mathbf{x})λ(x) on domain X\mathcal{X}X:

  1. Sample the total count N∼Poisson ⁣(∫Xλ(x) dx)N \sim \mathrm{Poisson}\!\left(\int_{\mathcal{X}} \lambda(\mathbf{x})\, d\mathbf{x}\right)N∼Poisson(∫X​λ(x)dx).

  2. Sample locations i.i.d. from the normalized intensity,

    xn∼iidλ(x)∫Xλ(x′) dx′.\mathbf{x}_n \overset{\text{iid}}{\sim} \frac{\lambda(\mathbf{x})}{\int_{\mathcal{X}} \lambda(\mathbf{x}')\, d\mathbf{x}'}.xn​∼iid∫X​λ(x′)dx′λ(x)​.
    (4)

This requires the ability to compute the integral of λ\lambdaλ and to sample from the normalized intensity.

2. Interval Sampling (Homogeneous Case)¶

A homogeneous Poisson process has constant intensity λ(x)≡λ\lambda(\mathbf{x}) \equiv \lambdaλ(x)≡λ.

Property 3: Inter-arrival times are i.i.d. exponential,

Δn=xn−xn−1∼iidExp(λ).\Delta_n = x_n - x_{n-1} \overset{\text{iid}}{\sim} \mathrm{Exp}(\lambda).Δn​=xn​−xn−1​∼iidExp(λ).
(5)

Property 4 (Memoryless): The waiting time until the next point is independent of time elapsed since the last point.

Algorithm: start at x0=0x_0 = 0x0​=0, repeatedly sample Δn∼Exp(λ)\Delta_n \sim \mathrm{Exp}(\lambda)Δn​∼Exp(λ) and set xn=xn−1+Δnx_n = x_{n-1} + \Delta_nxn​=xn−1​+Δn​, stopping when xn>Tx_n > Txn​>T.

3. Time-Rescaling (Inhomogeneous Case)¶

Define the cumulative intensity Λ(x)=∫0xλ(t) dt\Lambda(x) = \int_0^x \lambda(t)\, dtΛ(x)=∫0x​λ(t)dt. When λ>0\lambda > 0λ>0 everywhere, Λ\LambdaΛ is invertible.

Algorithm:

  1. Sample a homogeneous Poisson process with unit rate on [0,Λ(T)][0, \Lambda(T)][0,Λ(T)] to get U={un}\mathbf{U} = \{u_n\}U={un​}.

  2. Set X={Λ−1(un):un∈U}\mathbf{X} = \{\Lambda^{-1}(u_n) : u_n \in \mathbf{U}\}X={Λ−1(un​):un​∈U}.

This is the point-process analog of inverse-CDF sampling.

4. Thinning (Rejection Sampling)¶

Given a bounded intensity λ(x)≤λmax⁡\lambda(\mathbf{x}) \leq \lambda_{\max}λ(x)≤λmax​:

  1. Sample a homogeneous Poisson process with rate λmax⁡\lambda_{\max}λmax​ to get candidate points.

  2. Independently accept each candidate x\mathbf{x}x with probability λ(x)/λmax⁡\lambda(\mathbf{x}) / \lambda_{\max}λ(x)/λmax​.

This is the point-process analog of rejection sampling.

def sample_pp_topdown(intensity_fn, total_intensity, sample_location_fn, T):
    '''Sample an inhomogeneous Poisson process using the top-down method.

    Args:
        intensity_fn: callable, intensity at a point
        total_intensity: integral of intensity over [0, T]
        sample_location_fn: callable returning one sample from normalized intensity
        T: end of interval
    Returns:
        sorted array of event times
    '''
    N = torch.poisson(torch.tensor(total_intensity)).int().item()
    events = torch.tensor([sample_location_fn() for _ in range(N)])
    return events.sort().values


def sample_pp_intervals(lam, T):
    '''Sample a homogeneous Poisson process on [0, T] via inter-arrival intervals.

    Args:
        lam: rate (scalar)
        T: end of interval
    Returns:
        1D tensor of event times
    '''
    events = []
    t = 0.0
    while True:
        dt = torch.distributions.Exponential(torch.tensor(lam)).sample().item()
        t += dt
        if t > T:
            break
        events.append(t)
    return torch.tensor(events)


def sample_pp_thinning(intensity_fn, lam_max, T, dt=1e-3):
    '''Sample an inhomogeneous Poisson process on [0, T] via thinning.

    Args:
        intensity_fn: callable mapping time -> intensity (must be <= lam_max)
        lam_max: upper bound on intensity
        T: end of interval
        dt: time resolution (unused here; thinning uses continuous time)
    Returns:
        1D tensor of accepted event times
    '''
    candidates = sample_pp_intervals(lam_max, T)
    accepted = []
    for t in candidates:
        prob = intensity_fn(t.item()) / lam_max
        if torch.rand(1).item() < prob:
            accepted.append(t.item())
    return torch.tensor(accepted)

Superposition and Thinning¶

Property 5 (Superposition): The union of independent Poisson processes is also a Poisson process. If

{xn}n=1N∼PP(λ1(x))and{xm′}m=1M∼PP(λ2(x)),\{\mathbf{x}_n\}_{n=1}^N \sim \mathrm{PP}(\lambda_1(\mathbf{x})) \quad \text{and} \quad \{\mathbf{x}'_m\}_{m=1}^M \sim \mathrm{PP}(\lambda_2(\mathbf{x})),{xn​}n=1N​∼PP(λ1​(x))and{xm′​}m=1M​∼PP(λ2​(x)),
(6)

then their union {xn}∪{xm′}∼PP(λ1(x)+λ2(x))\{\mathbf{x}_n\} \cup \{\mathbf{x}'_m\} \sim \mathrm{PP}(\lambda_1(\mathbf{x}) + \lambda_2(\mathbf{x})){xn​}∪{xm′​}∼PP(λ1​(x)+λ2​(x)).

Poisson thinning is the reverse: given {xn}∼PP(λ1(x)+λ2(x))\{\mathbf{x}_n\} \sim \mathrm{PP}(\lambda_1(\mathbf{x}) + \lambda_2(\mathbf{x})){xn​}∼PP(λ1​(x)+λ2​(x)), independently assign each point to process 1 with probability λ1(xn)/(λ1(xn)+λ2(xn))\lambda_1(\mathbf{x}_n) / (\lambda_1(\mathbf{x}_n) + \lambda_2(\mathbf{x}_n))λ1​(xn​)/(λ1​(xn​)+λ2​(xn​)). The resulting subsets are independent Poisson processes with intensities λ1\lambda_1λ1​ and λ2\lambda_2λ2​ respectively.

Goodness-of-Fit Test via Time-Rescaling¶

Brown et al., 2002 developed a powerful goodness-of-fit test for inhomogeneous Poisson processes based on the time-rescaling theorem.

Procedure: Given observed points {xn}n=1N\{x_n\}_{n=1}^N{xn​}n=1N​ and a fitted intensity λ(x)\lambda(x)λ(x):

  1. Compute the rescaled inter-arrival intervals Δn=Λ(xn)−Λ(xn−1)\Delta_n = \Lambda(x_n) - \Lambda(x_{n-1})Δn​=Λ(xn​)−Λ(xn−1​) with Λ(x0)=0\Lambda(x_0) = 0Λ(x0​)=0. Under the correct model, Δn∼iidExp(1)\Delta_n \overset{\text{iid}}{\sim} \mathrm{Exp}(1)Δn​∼iidExp(1).

  2. Transform to uniform samples: zn=1−e−Δnz_n = 1 - e^{-\Delta_n}zn​=1−e−Δn​. Under the correct model, zn∼iidUniform([0,1])z_n \overset{\text{iid}}{\sim} \mathrm{Uniform}([0, 1])zn​∼iidUniform([0,1]).

  3. Sort to get order statistics z(1)≤⋯≤z(N)z_{(1)} \leq \cdots \leq z_{(N)}z(1)​≤⋯≤z(N)​.

  4. Plot (n/N, z(n))(n/N,\, z_{(n)})(n/N,z(n)​) — under the correct model this should fall near the 45° line.

Confidence bands: The nnn-th order statistic follows z(n)∼Beta(n,N−n+1)z_{(n)} \sim \mathrm{Beta}(n, N-n+1)z(n)​∼Beta(n,N−n+1). Use the 2.5% and 97.5% quantiles to build confidence bands.

def true_intensity(t):

def cum_intensity(t):
    # Lambda(t) = 2t - (5/pi)*cos(pi*t/5) + 5/pi
    return 2.0 * t - (5.0 / np.pi) * np.cos(np.pi * t / 5.0) + 5.0 / np.pi

# Sample events via thinning
lam_max_gof = 3.1

def gof_plot(ax, events, intensity_fn, cum_intensity_fn, T, title):
    '''KS / time-rescaling goodness-of-fit Q-Q plot.'''
    xs = np.sort(events)
    # compute Lambda differences
    xs_with_zero = np.concatenate([[0.0], xs])
    Lambda_vals = np.array([cum_intensity_fn(t) for t in xs_with_zero])
    Delta = np.diff(Lambda_vals)
    z = 1 - np.exp(-Delta)
    z_sorted = np.sort(z)
    n_pts = len(z_sorted)
    ns = np.arange(1, n_pts + 1)
    expected = (ns - 0.5) / n_pts
    # Beta confidence bands
    lower = beta_dist.ppf(0.025, ns, n_pts - ns + 1)
    upper = beta_dist.ppf(0.975, ns, n_pts - ns + 1)
    ax.fill_between(expected, lower, upper, alpha=0.3, color='tab:blue', label='95% CI')
    ax.plot([0, 1], [0, 1], 'k--', lw=1, label='45° line')
    ax.plot(expected, z_sorted, 'o', ms=3, color='tab:orange', label='Order stats')
    ax.set_xlabel('Expected quantile')
    ax.set_ylabel('Observed quantile')
    ax.set_title(title)
    ax.legend(fontsize=8)

Hawkes Processes¶

Hawkes processes Hawkes, 1971 are self-exciting point processes whose conditional intensity depends on the history of past events.

The conditional intensity is,

λ(t∣Ht)=λ0+∑tn∈Hth(t−tn),\lambda(t \mid \mathcal{H}_t) = \lambda_0 + \sum_{t_n \in \mathcal{H}_t} h(t - t_n),λ(t∣Ht​)=λ0​+tn​∈Ht​∑​h(t−tn​),
(7)

where h:R+→R+h: \mathbb{R}_+ \to \mathbb{R}_+h:R+​→R+​ is a non-negative impulse response (or influence function) and Ht={tn:tn<t}\mathcal{H}_t = \{t_n : t_n < t\}Ht​={tn​:tn​<t} is the history up to time ttt.

A common choice is an exponential impulse response,

h(Δt)=wτe−Δt/τ=w⋅Exp(Δt; τ),h(\Delta t) = \frac{w}{\tau} e^{-\Delta t / \tau} = w \cdot \mathrm{Exp}(\Delta t;\, \tau),h(Δt)=τw​e−Δt/τ=w⋅Exp(Δt;τ),
(8)

where τ>0\tau > 0τ>0 is the time constant and w>0w > 0w>0 is the total weight (i.e., the expected number of events triggered per event, ∫0∞h(Δt) d(Δt)=w\int_0^\infty h(\Delta t)\, d(\Delta t) = w∫0∞​h(Δt)d(Δt)=w).

Maximum Likelihood Estimation¶

The Hawkes log-likelihood has the same form as the Poisson process likelihood,

log⁡p({tn}∣θ)=−∫0Tλθ(t∣Ht) dt+∑n=1Nlog⁡λθ(tn∣Htn).\log p(\{t_n\} \mid \theta) = -\int_0^T \lambda_\theta(t \mid \mathcal{H}_t)\, dt + \sum_{n=1}^N \log \lambda_\theta(t_n \mid \mathcal{H}_{t_n}).logp({tn​}∣θ)=−∫0T​λθ​(t∣Ht​)dt+n=1∑N​logλθ​(tn​∣Htn​​).
(9)

With an exponential impulse response and parameters θ=(λ0,w)\theta = (\lambda_0, w)θ=(λ0​,w), this simplifies to,

log⁡p≈−θ⊤ϕ0+∑n=1Nlog⁡(θ⊤ϕn),\log p \approx -\theta^\top \phi_0 + \sum_{n=1}^N \log(\theta^\top \phi_n),logp≈−θ⊤ϕ0​+n=1∑N​log(θ⊤ϕn​),
(10)

where ϕ0=(T,N)⊤\phi_0 = (T, N)^\topϕ0​=(T,N)⊤ and ϕn=(1,  ∑tm<tnExp(tn−tm; τ))⊤\phi_n = \big(1,\; \sum_{t_m < t_n} \mathrm{Exp}(t_n - t_m;\, \tau)\big)^\topϕn​=(1,∑tm​<tn​​Exp(tn​−tm​;τ))⊤.

Multivariate Hawkes Processes¶

A multivariate Hawkes process Linderman & Adams, 2014 models SSS interacting sources with mutually excitatory interactions,

λs(t∣Ht)=λs,0+∑(tn,sn)∈Hthsn,s(t−tn),\lambda_s(t \mid \mathcal{H}_t) = \lambda_{s,0} + \sum_{(t_n, s_n) \in \mathcal{H}_t} h_{s_n, s}(t - t_n),λs​(t∣Ht​)=λs,0​+(tn​,sn​)∈Ht​∑​hsn​,s​(t−tn​),
(11)

where hs,s′(Δt)h_{s,s'}(\Delta t)hs,s′​(Δt) is a directed impulse response from source sss to source s′s's′.

The weights W\mathbf{W}W with Ws,s′=∫0∞hs,s′(Δt) d(Δt)W_{s,s'} = \int_0^\infty h_{s,s'}(\Delta t)\, d(\Delta t)Ws,s′​=∫0∞​hs,s′​(Δt)d(Δt) define a directed network of interactions.

Hawkes Processes as Poisson Clustering¶

By the superposition principle, the points from each source sss can be decomposed into clusters:

Ts=Ts,0∪⋃n=1NTs,n,\mathcal{T}_s = \mathcal{T}_{s,0} \cup \bigcup_{n=1}^N \mathcal{T}_{s,n},Ts​=Ts,0​∪n=1⋃N​Ts,n​,
(12)

where Ts,0∼PP(λs,0)\mathcal{T}_{s,0} \sim \mathrm{PP}(\lambda_{s,0})Ts,0​∼PP(λs,0​) are background events and Ts,n∼PP(hsn,s(t−tn))\mathcal{T}_{s,n} \sim \mathrm{PP}(h_{s_n, s}(t - t_n))Ts,n​∼PP(hsn​,s​(t−tn​)) are events triggered by the nnn-th event. With exponential impulse responses, E[∣Ts,n∣]=wsn,s\mathbb{E}[|\mathcal{T}_{s,n}|] = w_{s_n, s}E[∣Ts,n​∣]=wsn​,s​.

This clustering representation enables a Gibbs sampler for Bayesian inference: with a Gamma prior ws,s′∼Ga(α,β)w_{s,s'} \sim \mathrm{Ga}(\alpha, \beta)ws,s′​∼Ga(α,β), the weights and cluster assignments have conjugate conditionals that can be sampled alternately.

def sample_hawkes(lam0, w, tau, T, seed=0):
    '''Sample a univariate Hawkes process with exponential impulse response.

    Args:
        lam0: baseline intensity
        w: excitation weight (expected offspring per event)
        tau: time constant of exponential impulse response
        T: end of interval
        seed: random seed
    Returns:
        sorted numpy array of event times
    '''
    rng = np.random.default_rng(seed)
    events = []
    t = 0.0
    # Use Ogata's thinning algorithm
    while t < T:
        # Upper bound on intensity given current history
        lam_bar = lam0 + (w / tau) * sum(np.exp(-(t - ti) / tau) for ti in events)
        # Next candidate event
        dt = rng.exponential(1.0 / lam_bar)
        t_cand = t + dt
        if t_cand > T:
            break
        # Evaluate true intensity at candidate
        lam_cand = lam0 + (w / tau) * sum(np.exp(-(t_cand - ti) / tau) for ti in events)
        # Accept/reject
        if rng.uniform() < lam_cand / lam_bar:
            events.append(t_cand)
        t = t_cand
    return np.array(sorted(events))

Doubly Stochastic (Cox) Processes¶

Hawkes processes introduce dependence through the past. An alternative is to model the intensity itself as a random process — these are called doubly stochastic processes or Cox processes.

In these models,

λ(x)∼p(λ),\lambda(\mathbf{x}) \sim p(\lambda),λ(x)∼p(λ),
(13)

and then, given λ\lambdaλ, the points are generated as a Poisson process with that intensity.

Log Gaussian Cox process: Let f∼GP(μ(⋅),K(⋅,⋅))f \sim \mathrm{GP}(\mu(\cdot), K(\cdot, \cdot))f∼GP(μ(⋅),K(⋅,⋅)) and set λ(x)=ef(x)\lambda(\mathbf{x}) = e^{f(\mathbf{x})}λ(x)=ef(x). This gives a flexible, non-negative intensity with a log-Gaussian marginal distribution at each point.

Sigmoidal Gaussian Cox process Adams et al., 2009: Use λ(x)=σ(f(x))⋅λmax⁡\lambda(\mathbf{x}) = \sigma(f(\mathbf{x})) \cdot \lambda_{\max}λ(x)=σ(f(x))⋅λmax​ where σ\sigmaσ is the sigmoid function. This allows exact inference via a thinning construction.

Neyman–Scott process Wang et al., 2022: Let λ\lambdaλ be a convolution of a Poisson process with a non-negative kernel. This generates clustered point patterns where events arise in groups.

The key challenge in all these models is computing (or approximating) the intractable integral ∫λ(x) dx\int \lambda(\mathbf{x})\, d\mathbf{x}∫λ(x)dx in the likelihood.

Summary¶

  • Poisson processes are defined by an intensity function λ(x)\lambda(\mathbf{x})λ(x) and two properties: Poisson counts and independence of disjoint sets.

  • They can be sampled by four methods: top-down, intervals (homogeneous), time-rescaling, and thinning (rejection sampling).

  • The time-rescaling theorem yields a simple goodness-of-fit test: if the model is correct, the rescaled intervals are i.i.d. Exp(1)\mathrm{Exp}(1)Exp(1), and after a further transformation, the order statistics fall along a 45° line.

  • Hawkes processes introduce self-excitation via a conditional intensity that depends on past events; their clustering representation enables efficient Bayesian inference.

  • Cox processes introduce dependence by modeling the intensity as a stochastic process, typically a Gaussian process transformed to be non-negative.
