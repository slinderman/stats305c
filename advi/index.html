Gradient-Based VI

Prerequisites

This chapter builds on Coordinate Ascent Variational Inference. It extends variational inference to non-conjugate models using gradient estimators.

CAVI is powerful but requires the variational family to match the conjugate structure of the model. For non-conjugate models — Bayesian logistic regression, neural-network priors, physics-based simulators — no closed-form coordinate updates exist.

Gradient-based variational inference removes the conjugacy requirement by estimating the ELBO gradient with Monte Carlo and optimising with stochastic gradient ascent. This approach goes under several names: black-box VI (BBVI), automatic differentiation VI (ADVI), and fixed-form VI.

The central challenge is differentiating through an expectation:

∇λL(λ)=∇λEq(θ; λ)[log⁡p(x,θ)−log⁡q(θ;λ)].\nabla_{\mblambda} \mathcal{L}(\mblambda) = \nabla_{\mblambda} \mathbb{E}_{q(\mbtheta;\,\mblambda)} \bigl[\log p(\mbx, \mbtheta) - \log q(\mbtheta; \mblambda)\bigr].∇λ​L(λ)=∇λ​Eq(θ;λ)​[logp(x,θ)−logq(θ;λ)].
(1)

This chapter covers two gradient estimators that handle this:

  • Score function estimator (REINFORCE) — broadly applicable but high variance.

  • Pathwise gradient estimator (reparameterization trick) — lower variance, works whenever we can write θ=r(λ,ϵ)\mbtheta = r(\mblambda, \mbepsilon)θ=r(λ,ϵ) with ϵ\mbepsilonϵ independent of λ\mblambdaλ.

We conclude with a worked example: ADVI for Bayesian logistic regression, a non-conjugate model where CAVI has no closed-form updates.

Setup¶

Let θ\mbthetaθ denote all latent variables and parameters, and let Q={q(θ;λ):λ∈Λ}\mathcal{Q} = \{q(\mbtheta; \mblambda) : \mblambda \in \Lambda\}Q={q(θ;λ):λ∈Λ} be the variational family. We maximise the ELBO:

L(λ)=Eq(θ; λ) ⁣[log⁡p(x,θ)−log⁡q(θ;λ)]=log⁡p(x)−DKL(q(θ;λ)∥p(θ∣x)).\mathcal{L}(\mblambda) = \mathbb{E}_{q(\mbtheta;\,\mblambda)}\!\left[ \log p(\mbx, \mbtheta) - \log q(\mbtheta; \mblambda) \right] = \log p(\mbx) - D_{\mathrm{KL}}\bigl(q(\mbtheta;\mblambda) \| p(\mbtheta \mid \mbx)\bigr).L(λ)=Eq(θ;λ)​[logp(x,θ)−logq(θ;λ)]=logp(x)−DKL​(q(θ;λ)∥p(θ∣x)).
(2)

Strategy: assume Λ=RP\Lambda = \mathbb{R}^PΛ=RP (unconstrained parameters) and run (stochastic) gradient ascent:

λ←λ+αt ∇^λL(λ),\mblambda \leftarrow \mblambda + \alpha_t\, \widehat{\nabla}_{\mblambda} \mathcal{L}(\mblambda),λ←λ+αt​∇λ​L(λ),
(3)

where ∇^λL\widehat{\nabla}_{\mblambda} \mathcal{L}∇λ​L is an unbiased Monte Carlo estimate of the gradient.

The obstacle: the distribution under the expectation depends on λ\mblambdaλ, so we cannot simply move ∇λ\nabla_{\mblambda}∇λ​ inside the expectation.

The Score Function Estimator (REINFORCE)¶

Derivation¶

Use the log-derivative trick ∇λq=q⋅∇λlog⁡q\nabla_{\mblambda} q = q \cdot \nabla_{\mblambda} \log q∇λ​q=q⋅∇λ​logq to move the gradient inside the integral:

∇λEq(θ; λ)[h(θ)]=∫∇λq(θ;λ)⋅h(θ) dθ=Eq ⁣[∇λlog⁡q(θ;λ)⏟score⋅h(θ)].\nabla_{\mblambda} \mathbb{E}_{q(\mbtheta;\,\mblambda)}[h(\mbtheta)] = \int \nabla_{\mblambda} q(\mbtheta; \mblambda) \cdot h(\mbtheta)\,d\mbtheta = \mathbb{E}_{q}\!\left[\underbrace{\nabla_{\mblambda} \log q(\mbtheta; \mblambda)}_{\text{score}}\cdot h(\mbtheta)\right].∇λ​Eq(θ;λ)​[h(θ)]=∫∇λ​q(θ;λ)⋅h(θ)dθ=Eq​[score∇λ​logq(θ;λ)​​⋅h(θ)].
(4)

This gives the score function estimator (a.k.a. REINFORCE Williams, 1992):

∇^λL(λ)=1M∑m=1M[∇λlog⁡q(θ(m);λ)][log⁡p(x,θ(m))−log⁡q(θ(m);λ)],θ(m)∼iidq(θ;λ).\widehat{\nabla}_{\mblambda}\mathcal{L}(\mblambda) = \frac{1}{M}\sum_{m=1}^M \Bigl[\nabla_{\mblambda}\log q(\mbtheta^{(m)};\mblambda)\Bigr] \bigl[\log p(\mbx,\mbtheta^{(m)}) - \log q(\mbtheta^{(m)};\mblambda)\bigr], \quad \mbtheta^{(m)} \overset{\text{iid}}{\sim} q(\mbtheta;\mblambda).∇λ​L(λ)=M1​m=1∑M​[∇λ​logq(θ(m);λ)][logp(x,θ(m))−logq(θ(m);λ)],θ(m)∼iidq(θ;λ).
(5)

Properties:

  • Works for discrete and continuous θ\mbthetaθ — only requires that log⁡q\log qlogq is differentiable wrt λ\mblambdaλ.

  • Often has high variance in practice.

Variance Reduction: Control Variates¶

Since the score has zero expectation, Eq[∇λlog⁡q(θ;λ)]=0\mathbb{E}_q[\nabla_{\mblambda} \log q(\mbtheta; \mblambda)] = 0Eq​[∇λ​logq(θ;λ)]=0, we can subtract any baseline bbb without changing the expectation:

∇λL=Eq ⁣[(h(θ)−b) ∇λlog⁡q(θ;λ)].\nabla_{\mblambda}\mathcal{L} = \mathbb{E}_q\!\bigl[(h(\mbtheta) - b)\,\nabla_{\mblambda}\log q(\mbtheta;\mblambda)\bigr].∇λ​L=Eq​[(h(θ)−b)∇λ​logq(θ;λ)].
(6)

Choosing b=Eq[h(θ)]b = \mathbb{E}_q[h(\mbtheta)]b=Eq​[h(θ)] (the running mean of hhh) can reduce variance substantially.

The Pathwise Gradient Estimator (Reparameterisation Trick)¶

Derivation¶

If we can write θ=r(λ,ϵ)\mbtheta = r(\mblambda, \mbepsilon)θ=r(λ,ϵ) where ϵ\mbepsilonϵ has a distribution independent of λ\mblambdaλ (a reparameterisation), then:

Eq(θ; λ)[h(θ,λ)]=Eϵ∼p(ϵ)[h(r(λ,ϵ),λ)].\mathbb{E}_{q(\mbtheta;\,\mblambda)}[h(\mbtheta, \mblambda)] = \mathbb{E}_{\mbepsilon \sim p(\mbepsilon)}[h(r(\mblambda, \mbepsilon), \mblambda)].Eq(θ;λ)​[h(θ,λ)]=Eϵ∼p(ϵ)​[h(r(λ,ϵ),λ)].
(7)

The gradient now passes through the expectation cleanly:

∇λEq[h(θ,λ)]=Eϵ ⁣[∇λh(r(λ,ϵ),λ)].\nabla_{\mblambda} \mathbb{E}_{q}[h(\mbtheta, \mblambda)] = \mathbb{E}_{\mbepsilon}\!\left[\nabla_{\mblambda} h(r(\mblambda, \mbepsilon), \mblambda)\right].∇λ​Eq​[h(θ,λ)]=Eϵ​[∇λ​h(r(λ,ϵ),λ)].
(8)

Monte Carlo estimate:

∇^λL(λ)=1M∑m=1M∇λ[log⁡p(x,r(λ,ϵm))−log⁡q(r(λ,ϵm);λ)],ϵm∼iidp(ϵ).\widehat{\nabla}_{\mblambda}\mathcal{L}(\mblambda) = \frac{1}{M}\sum_{m=1}^M \nabla_{\mblambda}\bigl[\log p(\mbx, r(\mblambda, \mbepsilon_m)) - \log q(r(\mblambda, \mbepsilon_m); \mblambda)\bigr], \quad \mbepsilon_m \overset{\text{iid}}{\sim} p(\mbepsilon).∇λ​L(λ)=M1​m=1∑M​∇λ​[logp(x,r(λ,ϵm​))−logq(r(λ,ϵm​);λ)],ϵm​∼iidp(ϵ).
(9)

In practice: just call .backward() on a Monte Carlo ELBO estimate computed with reparameterised samples; PyTorch handles the chain rule automatically.

Example: Diagonal Gaussian¶

For q(θ;μ,σ)=N(θ;μ,diag(σ2))q(\mbtheta; \mbmu, \mbsigma) = \mathcal{N}(\mbtheta; \mbmu, \mathrm{diag}(\mbsigma^2))q(θ;μ,σ)=N(θ;μ,diag(σ2)):

θ=r(μ,σ,ϵ)=μ+σ⊙ϵ,ϵ∼N(0,I).\mbtheta = r(\mbmu, \mbsigma, \mbepsilon) = \mbmu + \mbsigma \odot \mbepsilon, \qquad \mbepsilon \sim \mathcal{N}(\mathbf{0}, \mathbf{I}).θ=r(μ,σ,ϵ)=μ+σ⊙ϵ,ϵ∼N(0,I).
(10)

Non-reparameterisable families: Discrete distributions (Bernoulli, Categorical) cannot be reparameterised in the standard sense. For these, the score function estimator (or straight-through / Gumbel-softmax approximations) must be used.

# ── Variance comparison: score function vs pathwise estimator ─────────────
# Model: observe x=5 from N(theta, 1) with prior N(0, 10^2)
# True posterior: N(mu_post, sigma_post^2) by Bayes rule
# Variational: q(theta; mu, log_sigma) = N(mu, sigma^2)

x_obs    = torch.tensor(5.0)
prior_mu, prior_sig = torch.tensor(0.0), torch.tensor(10.0)
lik_sig  = torch.tensor(1.0)

# True posterior  (Gaussian conjugate)
sigma_post = 1.0 / (1/lik_sig**2 + 1/prior_sig**2).sqrt()
mu_post    = sigma_post**2 * (x_obs/lik_sig**2 + prior_mu/prior_sig**2)
print(f'True posterior: N({mu_post.item():.3f}, {sigma_post.item():.3f}²)')

def log_joint(theta):
    return (dist.Normal(theta, lik_sig).log_prob(x_obs)
            + dist.Normal(prior_mu, prior_sig).log_prob(theta))

def elbo_and_grads(mu, log_sigma, M, estimator='pathwise'):
    sigma = log_sigma.exp()
    if estimator == 'pathwise':
        # Reparameterise: theta = mu + sigma * eps
        eps   = torch.randn(M)
        theta = mu + sigma * eps
        lp    = log_joint(theta)
        lq    = dist.Normal(mu, sigma).log_prob(theta)
        elbo  = (lp - lq).mean()
        elbo.backward()
        g_mu, g_ls = mu.grad.clone(), log_sigma.grad.clone()
    else:  # score function
        with torch.no_grad():
            theta = dist.Normal(mu, sigma).rsample((M,))
            lp    = log_joint(theta)
            lq    = dist.Normal(mu, sigma).log_prob(theta)
            h     = lp - lq                             # (M,)
            # score wrt mu: (theta - mu) / sigma^2
            score_mu = (theta - mu) / sigma**2
            # score wrt log_sigma: (theta-mu)^2/sigma^2 - 1
            score_ls = (theta - mu)**2 / sigma**2 - 1.0
            g_mu  = (h * score_mu).mean()
            g_ls  = (h * score_ls).mean()
            elbo  = h.mean()
    return elbo.detach(), g_mu, g_ls

mu_val, ls_val = mu_post.item(), sigma_post.log().item()

ADVI for Bayesian Logistic Regression¶

Model¶

Consider NNN binary observations yn∈{0,1}y_n \in \{0,1\}yn​∈{0,1} with DDD-dimensional features xn\mbx_nxn​. A logistic regression model with Gaussian prior is:

w∼N(0,τ2I),yn∣w,xn∼Bernoulli⁡ ⁣(σ(w⊤xn)),\mbw \sim \mathcal{N}(\mathbf{0}, \tau^2 \mathbf{I}), \qquad y_n \mid \mbw, \mbx_n \sim \operatorname{Bernoulli}\!\bigl(\sigma(\mbw^\top \mbx_n)\bigr),w∼N(0,τ2I),yn​∣w,xn​∼Bernoulli(σ(w⊤xn​)),
(11)

where σ(a)=1/(1+e−a)\sigma(a) = 1/(1+e^{-a})σ(a)=1/(1+e−a) is the sigmoid. The posterior p(w∣X,y)p(\mbw \mid \mbX, \mby)p(w∣X,y) has no closed form (the likelihood is not conjugate to the Gaussian prior), making CAVI inapplicable.

Variational Family¶

Use a mean-field Gaussian:

q(w;μ,ρ)=N ⁣(w;  μ,  diag⁡(e2ρ)),q(\mbw; \mbmu, \boldsymbol{\rho}) = \mathcal{N}\!\bigl(\mbw;\; \mbmu,\; \operatorname{diag}(e^{2\boldsymbol{\rho}})\bigr),q(w;μ,ρ)=N(w;μ,diag(e2ρ)),
(12)

where μ,ρ∈RD\mbmu, \boldsymbol{\rho} \in \mathbb{R}^Dμ,ρ∈RD are unconstrained variational parameters (σ=eρ\mbsigma = e^{\boldsymbol{\rho}}σ=eρ enforces positivity).

ELBO¶

L(μ,ρ)=Eq ⁣[∑n=1Nlog⁡σ(yn′w⊤xn)]⏟expected log-likelihood (MC)−DKL ⁣(N(μ,diag⁡(e2ρ))∥N(0,τ2I))⏟closed form,\mathcal{L}(\mbmu, \boldsymbol{\rho}) = \underbrace{\mathbb{E}_{q}\!\left[\sum_{n=1}^N \log \sigma(y_n' \mbw^\top \mbx_n)\right]}_{\text{expected log-likelihood (MC)}} - \underbrace{D_{\mathrm{KL}}\!\bigl(\mathcal{N}(\mbmu, \operatorname{diag}(e^{2\boldsymbol{\rho}})) \| \mathcal{N}(\mathbf{0}, \tau^2 \mathbf{I})\bigr)}_{\text{closed form}},L(μ,ρ)=expected log-likelihood (MC)Eq​[n=1∑N​logσ(yn′​w⊤xn​)]​​−closed formDKL​(N(μ,diag(e2ρ))∥N(0,τ2I))​​,
(13)

where yn′=2yn−1∈{−1,+1}y_n' = 2y_n - 1 \in \{-1, +1\}yn′​=2yn​−1∈{−1,+1}. We estimate the expected log-likelihood via a single reparameterised sample (M=1M=1M=1) per gradient step, relying on Adam to average out the noise.

# ── Synthetic binary classification data ─────────────────────────────────────
N, D = 200, 4
w_true = torch.tensor([2.0, -1.5, 0.5, -0.8])
X = torch.randn(N, D)
y = dist.Bernoulli(logits=X @ w_true).sample()
print(f'Data: N={N}, D={D}, fraction positive = {y.mean():.2f}')

# ── ADVI with pathwise gradient estimator (Adam) ──────────────────────────────
def advi_logistic(X, y, τ=2.0, M=1, lr=0.05, num_iters=1000, seed=0):
    """ADVI for Bayesian logistic regression.

    Variational family: q(w) = N(μ, diag(exp(2ρ)))
    Uses the reparameterisation trick + PyTorch autograd.

    Parameters
    ----------
    τ   : prior standard deviation
    M   : MC samples per gradient step
    """
    torch.manual_seed(seed)
    N, D = X.shape

    μ = torch.zeros(D, requires_grad=True)
    ρ = torch.full((D,), -1.0, requires_grad=True)   # log σ, init σ ≈ 0.37

    optimiser = torch.optim.Adam([μ, ρ], lr=lr)
    elbos = []

    for _ in range(num_iters):
        optimiser.zero_grad()

        σ   = ρ.exp()                                # (D,)

        # Reparameterised samples: w = μ + σ * ε,  ε ~ N(0,I)
        ε   = torch.randn(M, D)
        w   = μ.unsqueeze(0) + σ.unsqueeze(0) * ε   # (M, D)

        # Expected log-likelihood  (averaged over M samples and N data points)
        logits   = X @ w.T                           # (N, M)
        log_lik  = dist.Bernoulli(logits=logits).log_prob(y.unsqueeze(1))
        ell      = log_lik.sum(0).mean()             # scalar

        # KL[ N(μ, diag(σ²)) || N(0, τ²I) ] — closed form
        kl = dist.kl_divergence(
            dist.Normal(μ, σ),
            dist.Normal(torch.zeros(D), τ * torch.ones(D))
        ).sum()

        loss = -(ell - kl)                           # minimise negative ELBO
        loss.backward()
        optimiser.step()

        elbos.append(-loss.item())

    return μ.detach(), ρ.exp().detach(), elbos


μ_vi, σ_vi, elbos = advi_logistic(X, y, τ=2.0, M=1, lr=0.05, num_iters=1000)

Mini-Batches and Stochastic Optimisers¶

Mini-Batch ELBO¶

When NNN is large, each gradient step over the full dataset is expensive. We can subsample a mini-batch B⊂{1,…,N}\mathcal{B} \subset \{1,\ldots,N\}B⊂{1,…,N} and scale up the log-likelihood:

L(λ)=N∣B∣∑n∈BEq[log⁡p(xn∣θ)]−DKL(q∥p),\mathcal{L}(\mblambda) = \frac{N}{|\mathcal{B}|}\sum_{n \in \mathcal{B}} \mathbb{E}_{q}[\log p(x_n \mid \mbtheta)] - D_{\mathrm{KL}}(q \| p),L(λ)=∣B∣N​n∈B∑​Eq​[logp(xn​∣θ)]−DKL​(q∥p),
(14)

giving an unbiased estimate of the ELBO and its gradient.

Stochastic Optimisers¶

Basic SGD often converges slowly. Several adaptive-step methods work well for ELBO optimisation:

MethodUpdate ruleNotes
SGD Robbins & Siegmund, 1971λ←λ+αg^\mblambda \leftarrow \mblambda + \alpha \hat{g}λ←λ+αg^​Simple; needs careful α\alphaα schedule
AdaGrad Duchi et al., 2011Per-parameter α/∑g^2\alpha / \sqrt{\sum \hat{g}^2}α/∑g^​2​Good for sparse gradients
Adam Kingma & Ba, 2014Momentum + adaptive scaleDefault choice; robust

Robbins-Monro conditions for SGD convergence: the step sizes must satisfy ∑tαt=∞\sum_t \alpha_t = \infty∑t​αt​=∞ and ∑tαt2<∞\sum_t \alpha_t^2 < \infty∑t​αt2​<∞.

Adam is the practical default for ADVI — it rarely requires step-size tuning and handles the varying scales of μ\muμ and ρ\rhoρ automatically.

Conclusion¶

Gradient-based VI removes the conjugacy requirement of CAVI, enabling variational inference in any model where the log joint is differentiable.

Key takeaways:

  • The log-derivative trick converts ∇λEq[h]\nabla_{\mblambda} \mathbb{E}_q[h]∇λ​Eq​[h] into an expectation computable by MC — but with high variance.

  • The reparameterisation trick decouples sampling noise from λ\mblambdaλ, moving the gradient inside the expectation and drastically reducing variance.

  • In PyTorch, reparameterised VI is nearly automatic: construct the ELBO using .rsample(), call .backward(), step an optimiser.

  • Adam is the practical default; a single MC sample per step (M=1M=1M=1) usually suffices when paired with an adaptive optimiser.

  • The KL term between two Gaussians (or other conjugate pairs) can be computed analytically, reducing MC noise further.
